````
A, N = map(int, input().split())
for i in range(N):
    A += N  # Add N to A and reassign the new value to A
    print(A)
````
`A+N`으로 작성하게 되면 값 고정으로 내려오게됨
=> A값에 N만큼 증가하게 하려면 +=으로 작성,
위에 for 문으로 N 만큼 반복되므로 변동된 A값이 N만큼 출력되게 됨

## FOR
Q. 자연수 n의 배수 5개를 작은 수부터 차례로 공백을 사이에 두고 출력 (1<=N<=100)
````
# 처음 작성 코드
N = int(input())
for i in range(5):
    N+=N
    print(N, end=' ') 
````
배수 출력이 아니라 값을 계속 누적변경 하게 됨
````
# 정답
N = int(input())
for i in range(1, 6):
    print(N*i, end=' ')
````
N은 변경하지 않고 i 만 1->5까지 변경

````
# n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.
# 1부터 n까지 합을 출력한다.
n = int(input())
total = 0 #총합 0으로 설정
for i in range(1,n+1): #1부터 stop 범위 n+1까지
    total += i #총합에 i 하나씩 더하기
print(total)
````
